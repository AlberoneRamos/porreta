porreta

Get funny words in Brazilian Portuguese.

"porreta" is a word commonly used on the northeast of Brazil to indicate something that is awesome.

Install

To install porreta using npm

                $ npm install porreta

Usage

Usage of porreta

                
var porreta = require('porreta');

porreta(); // pamonha
porreta(); // cangote

porreta.words; // ['alpendre', 'arapuca', 'araraquara', ...]

API

porreta()

Type: string
Random funny word.

porreta.words

Type: array
All the words.
